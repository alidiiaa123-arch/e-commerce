I’m excited to share my latest project — a modern **Clothing E-commerce Web App** built using React, Next.js, Tailwind CSS, and shadcn/ui.

🔗 Live Demo:
https://freshcart-xi-orpin.vercel.app/

🎨 UI Highlights:

* Built with shadcn/ui for reusable components
* Consistent and scalable design system
